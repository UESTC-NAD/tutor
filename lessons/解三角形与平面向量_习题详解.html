一、选择与填空（11–14）余弦定理 · 向量平行的坐标条件 · 锐角三角形取值范围

11
货轮在 A 处看灯塔 B 在北偏东 75°、距离 12√6 n mile；看灯塔 C 在北偏西 30°、距离 8√3 n mile。货轮由 A 向正北航行到 D 处时，看灯塔 B 在南偏东 60° 方向。下列说法正确的是（　）多选·方位角
A. A 与 D 间距离为 24 n mile　　B. 灯塔 C 与 D 间距离为 16 n mile
C. 灯塔 C 在 D 的西偏南 60° 方向　　D. D 在灯塔 B 的北偏西 30° 方向
解析

判断 A：在 △ABD 中，AD 为正北。∠DAB=75°（北与北偏东 75° 之夹角）；D 处看 B 为南偏东 60°，DA 指向正南，故 ∠ADB=60°；从而 ∠ABD=180°−75°−60°=45°。

AD = 12√6 · sin45°sin60° = 12√6 · √2⁄2√3⁄2 = 12√6 · √2√3 = 24　→ A 正确

判断 B、C：在 △ACD 中，AC=8√3，AD=24，∠CAD=30°。

CD2 = AC2+AD2−2·AC·AD·cos30° = 192+576−576 = 192
CD = √192 = 8√3 ≈ 13.9 ≠ 16　→ B 错误

又 AC=CD=8√3，△ACD 等腰，∠ADC=∠CAD=30°。DA 指向正南、C 在西侧，故 C 在 D 的南偏西 30°，即 西偏南 60° → C 正确。

判断 D：D 处看 B 为南偏东 60°，反向即 B 处看 D 为北偏西 60°（非 30°）→ D 错误。

答案：A C
12
在 △ABC 中，a·sinA+b·sinB=c·sinC−√3·b·sinA，则 C= ____。余弦定理
解析

由正弦定理（sinA : sinB : sinC = a : b : c），两边同乘 2R：

a2+b2=c2−√3·ab　⇒　a2+b2−c2=−√3·ab
cosC = a2+b2−c22ab = −√3·ab2ab = −√32
答案：C = 5π6（即 150°）
13
△ABC 中，m=(a+b, sinC)，n=(√3·a+c, sinB−sinA)，若 m∥n，求角 B。向量平行
解析

由 m∥n 得 (a+b)(sinB−sinA)−sinC·(√3·a+c)=0，用正弦定理化边：

(a+b)(b−a)−c(√3·a+c)=0　⇒　b2−a2=√3·ac+c2
a2+c2−b2=−√3·ac　⇒　cosB = a2+c2−b22ac = −√32
答案：B = 5π6（即 150°）
14
锐角 △ABC 中，a=1，B=2A，求 b 的取值范围。取值范围
解析

由正弦定理 b = a·sinBsinA = sin2AsinA = 2cosA。三个内角 A、2A、π−3A 均为锐角：

  • 2A < π2　⇒　A < π4；
  • π−3A < π2　⇒　A > π6。

故 A ∈ (π6, π4)，cosA ∈ (√22, √32)，b=2cosA ∈ (√2, √3)。

答案：b ∈ ( √2 , √3 )

二、解答题（15–19）向量运算 · 解三角形综合 · 方位角实际应用 · 中线与面积

15
（13 分）已知平面向量 a=(1, x)，b=(2x+3, −x)，x∈R。（1）若 a⊥b，求 x；（2）若 a∥b，求 |a−b|。平面向量
解析

（1）垂直：a·b=0。

1·(2x+3)+x·(−x)=0 ⇒ x2−2x−3=0 ⇒ (x−3)(x+1)=0

⇒ x=3 或 x=−1。

（2）平行：1·(−x)−x·(2x+3)=0 ⇒ −2x2−4x=0 ⇒ x=0 或 x=−2。

  • x=0：a=(1, 0)，b=(3, 0)，a−b=(−2, 0)，|a−b|=2；
  • x=−2：a=(1, −2)，b=(−1, 2)，a−b=(2, −4)，|a−b|=√(4+16)=2√5。
答案：（1）x=3 或 −1；（2）|a−b|=2 或 2√5
16
（15 分）四边形 ABCD 中，AD∥BC，AD=6，BC=4，CD=2，∠CBD=30°。（1）求 BD；（2）求 A。解三角形
解析

（1）求 BD：在 △BCD 中，BC=4，CD=2，∠CBD=30°，由正弦定理：

sin∠BDC = BC·sin30°CD = 4×½2 = 1　⇒　∠BDC=90°
BD = √(BC2−CD2) = √(16−4) = 2√3

（2）求 A：因 AD∥BC、BD 为截线，故 ∠ADB=∠DBC=30°。在 △ABD 中，AD=6，BD=2√3，∠ADB=30°：

AB2 = 36+12−2·6·2√3·√32 = 48−36 = 12

即 AB=2√3=BD，△ABD 等腰，∠A=∠ADB=30°。

答案：（1）BD=2√3；（2）A=π6（30°）
17
（15 分）锐角 △ABC 中，a、b、c 为角 A、B、C 的对边，m=(2b−c, cosC)，n=(a, cosA)，且 m∥n。（1）求角 A；（2）求 y=2sin2B+cos(π3−2B) 的值域。向量+三角
解析

（1）求 A：由 m∥n：(2b−c)cosA−a·cosC=0，正弦定理化角：

2sinB·cosA = sinA·cosC+cosA·sinC = sin(A+C) = sinB

因 sinB≠0，得 cosA=½，A=π3。

（2）求值域：用 2sin2B=1−cos2B，展开 cos(π3−2B)=½cos2B+√32sin2B：

y = 1−cos2B+½cos2B+√32sin2B = 1+√32sin2B−½cos2B = 1+sin(2B−π6)

锐角三角形且 A=π3：B<π2 且 C=2π3−B<π2 ⇒ B>π6，即 B∈(π6, π2)。

2B−π6 ∈ (π6, 5π6)　⇒　sin(2B−π6) ∈ ( ½ , 1 ]
答案：（1）A=π3；（2）y∈( 32 , 2 ]
18
（17 分）A、B 为东西方向相距 (3+√3) n mile 的两观测点。渔船 D 位于 A 的北偏东 60°、B 的西北（北偏西 45°）方向。救援船 C 位于 B 的南偏西 75°、距 B 为 3√6 n mile 处，速度 30 n mile/h。（1）求 B 到渔船 D 的距离；（2）C 处救援船到达 D 需多长时间？方位角应用
解析

（1）求 BD：在 △ABD 中，AB 沿正东：∠DAB=90°−60°=30°，∠DBA=90°−45°=45°，故 ∠ADB=105°。由正弦定理（sin105°=√6+√24）：

BD = AB·sin30°sin105° = 2(3+√3)√6+√2 = √6

（2）求 CD 及时间：BD 在北偏西 45°，BC 在南偏西 75°，夹角 ∠DBC=45°+15°=60°。在 △BCD 中，BD=√6，BC=3√6：

CD2 = BD2+BC2−2·BD·BC·cos60° = 6+54−18 = 42
CD = √42，　t = CD30 = √4230 h ≈ 0.22 h ≈ 13 min
答案：（1）BD=√6 n mile；（2）CD=√42 n mile，用时 √4230 小时（约 13 分钟）
19
（17 分·2023 新课标Ⅱ卷）△ABC 内角 A、B、C 的对边为 a、b、c，面积为 √3，D 为 BC 中点，AD=1。（1）若 ∠ADC=π3，求 tanB；（2）若 b2+c2=8，求 b、c。中线与面积
解析

（1）求 tanB：D 为中点，S△ABD=½S△ABC=√32；∠ADB=π−π3=2π3。

½·AD·BD·sin2π3 = √32 ⇒ ½·1·BD·√32 = √32 ⇒ BD=2

在 △ABD 中，AD=1，BD=2，∠ADB=2π3：

AB2 = 1+4−2·1·2·cos2π3 = 5+2 = 7 ⇒ AB=√7
sinB = AD·sin∠ADBAB = √3⁄2√7 = √2114，　cosB = 7+4−14√7 = 5√714

⇒ tanB = sinB / cosB = √35。

（2）求 b、c：中线长公式 AD2=2b2+2c2−a24，代入 AD=1、b2+c2=8：

4 = 2·8−a2 ⇒ a2=12

由面积与余弦定理：cosA=b2+c2−a22bc=−2bc，sinA=2Sbc=2√3bc。代入 sin2A+cos2A=1：

12+4(bc)2 = 1 ⇒ bc=4

结合 b2+c2=8=2bc，得 (b−c)2=0，b=c=2。

答案：（1）tanB=√35；（2）b=c=2
